# **App Name**: Profolio

## Core Features:

- User Authentication: Allow users to sign up, sign in, and manage their accounts securely to access their saved CVs.
- CV Data Input Form: An intuitive and interactive form for users to input all relevant CV information, including personal details, experience, education, and skills.
- Real-time CV Preview: Display a live, interactive preview of the CV as it's being built, updating instantly with every input.
- AI-Powered Bullet Point Enhancement Tool: A tool that provides suggestions and refinements for experience and skill bullet points to enhance clarity, impact, and keyword optimization.
- Save CV Data: Enable users to save their CV data securely to their personal profile for future editing and generation.
- Dashboard for Saved CVs: A personalized dashboard where users can view and manage a list of all their previously created and saved CVs.
- CV Template Application: Apply selected professional templates, dynamically populating them by replacing '{{placeholder}}' tokens with user-provided CVData values.

## Style Guidelines:

- The app utilizes a light color scheme to promote clarity and readability, aligning with the professional nature of a CV maker. The primary color is a deep, professional blue (#333399), providing a sense of trust and reliability. The background color is a subtle, light lavender-gray (#EDEDF4), providing a clean canvas without distraction. An accent of vibrant sky blue (#50C0EB) is used for calls to action and important highlights, offering a refreshing and modern touch while maintaining professionalism.
- The typeface 'Inter' (sans-serif) is chosen for its modern, clean, and highly readable characteristics, making it suitable for both headlines and extensive body text in forms and generated CVs.
- Icons should be clean, line-based, and minimalistic to maintain a professional and uncluttered aesthetic that complements the overall design.
- The main editor features a split-pane layout, presenting the input form and live preview side-by-side for an efficient user workflow. Overall layout prioritizes responsiveness and clear hierarchical organization of content.
- Subtle animations are incorporated for smooth transitions, form feedback (e.g., validation), and progress indicators, enhancing the user experience without being distracting.
